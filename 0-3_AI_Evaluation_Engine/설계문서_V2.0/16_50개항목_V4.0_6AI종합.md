## 📈 평가 방법론

### Bayesian Prior 6.0 점수 계산

```python
def calculate_bayesian_score_v5(scores, count):
    """
    V5.0: Prior 6.0, 편차 ±4

    - 개별 데이터: -10 ~ +10 범위
    - 총합 범위: -10N ~ +10N
    - 편차: (총합 × 4) / (10 × N)
    - 최종 점수: 6.0 + 편차
    - 절대 범위: 2.0 ~ 10.0
    """
    if count == 0:
        return 6.0

    total_sum = sum(scores)
    deviation = (total_sum * 4) / (10 * count)
    final_score = 6.0 + deviation
    final_score = max(2.0, min(10.0, final_score))

    return final_score
```
