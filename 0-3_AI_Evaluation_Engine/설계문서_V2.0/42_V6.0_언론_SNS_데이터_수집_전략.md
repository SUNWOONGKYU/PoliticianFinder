## 문서 개요

V6.0 프레임워크는 100% 정형 데이터로 구성되어 있으나, 다음과 같은 목적으로 **언론·SNS·여론조사** 등 비정부 데이터를 보조적으로 활용할 필요가 있습니다:

### 비정부 데이터의 활용 목적

1. **교차 검증 (Cross-Verification)**: 공공 데이터의 정확성 및 완전성 검증
2. **데이터 갭 보완 (Gap Filling)**: 공공 데이터가 미비하거나 누락된 경우 보완
3. **실시간 모니터링 (Real-time Monitoring)**: 공공 기록 반영 전 선제적 정보 수집
4. **맥락 이해 (Contextual Understanding)**: 정량 데이터의 배경 및 사회적 평가 파악
5. **예비 스크리닝 (Preliminary Screening)**: 본격적 조사 전 의심 사례 사전 식별

### 법적 준수 원칙

- **확정 사실만 활용**: 의혹, 추정, 논란 등 불확정 표현은 배제
- **공식 보도 우선**: 주요 언론사의 팩트체크된 기사 우선
- **저작권 준수**: 뉴스 기사는 제목·요약·링크만 저장, 전문 저장 금지
- **개인정보 보호**: SNS 수집 시 공개된 정보만 활용, 비공개 계정 제외
- **명예훼손 방지**: 법원 판결, 공식 발표 등 확정 사실 기반

## 데이터 유형별 활용 분류

### A. 언론 데이터 필수 항목 (Media-Essential Items)
공공 데이터만으로 수집이 어렵거나 불완전하여 **언론 보도가 필수적인 항목**

### B. SNS 데이터 필수 항목 (SNS-Essential Items)
의원의 자발적 공개나 시민 소통 측정에 **SNS 데이터가 필수적인 항목**

### C. 공공+언론 병행 항목 (Hybrid Items)
공공 데이터가 주 출처이나, **언론 보도로 교차 검증 및 보완이 유용한 항목**

### D. 여론조사 활용 항목 (Public Opinion Items)
정량 데이터로 측정 어려운 **시민 체감 평가를 여론조사로 보완 가능한 항목**

### 분야 1: 전문성 (5개 항목)

**언론/SNS 필요성**: 낮음
**활용 전략**:

- 언론 활용: 학력 위조 의혹 보도 시 교차 검증용
- 키워드: "[의원명] + 학력 위조", "[의원명] + 학위 논란"
- 수집 출처: 네이버 뉴스, 다음 뉴스 (팩트체크 기사)
- 법적 고려: 확정 판결 또는 대학 공식 발표만 인용

#### 1-2. 전공-직무 관련성 점수
**데이터 유형**: 공공 데이터 충분
**언론/SNS 필요성**: 낮음
**활용 전략**: 언론 불필요 (선관위 전공 정보 + 의회 위원회 배정으로 충분)

#### 1-3. 관련 분야 경력 연수
**데이터 유형**: 공공+언론 병행 권장
**언론/SNS 필요성**: 중간
**활용 전략**:
- 주 출처: 중앙선거관리위원회 (후보자 경력)
- 언론 활용: 경력 허위 기재 의혹 검증
- 키워드: "[의원명] + 경력 위조", "[의원명] + 허위 경력"
- 수집 출처: 네이버 뉴스, 조선일보, 중앙일보, 한겨레
- 수집 방법: 의원명 + 경력 키워드 검색
- 법적 고려: 선관위 확정 자료와 언론 보도 교차 검증

#### 1-4. 전문 자격증 보유 개수
**데이터 유형**: 공공+언론 병행
**언론/SNS 필요성**: 중간
**활용 전략**:
- 주 출처: 선관위 + 각 협회 조회 (변호사협회, 공인회계사회 등)
- 언론 활용: 자격증 위조·대여 의혹 검증
- 키워드: "[의원명] + 자격증 위조", "[의원명] + 변호사 자격"
- 수집 출처: 법률신문, 조세일보 등 전문지 포함
- 법적 고려: 협회 공식 확인 또는 법원 판결 기반

#### 1-5. 연간 직무 교육 이수 시간
**데이터 유형**: 공공 데이터 충분 (정보공개청구)
**언론/SNS 필요성**: 낮음
**활용 전략**: 언론 불필요 (의회 사무처 정보공개청구로 충분)

**분야 1 요약**: 언론 활용 2개 (경력, 자격증 검증용)

### 분야 4: 청렴성 (5개 항목)

#### 4-1. 부패 범죄 확정 판결
**데이터 유형**: 공공+언론 병행 필수
**언론/SNS 필요성**: 높음
**활용 전략**:
- 주 출처: 대법원 판결문 공개시스템
- 언론 활용: 판결문 발견 및 확정 판결 확인
- 키워드:
  - "[의원명] + 뇌물 + 판결"
  - "[의원명] + 횡령 + 유죄"
  - "[의원명] + 배임 + 확정"
  - "[의원명] + 알선수재 + 법원"
- 수집 출처:
  - **주요 일간지**: 조선일보, 중앙일보, 동아일보, 한겨레, 경향신문
  - **법률 전문지**: 법률신문, 법조신문
  - **뉴스 포털**: 네이버 뉴스, 다음 뉴스, 구글 뉴스
- 수집 방법:
  - 뉴스 검색 API (네이버 뉴스 API, 다음 검색 API)
  - 키워드 조합 검색 (의원명 + 범죄명 + "판결" OR "유죄" OR "확정")
  - 수집 기간: 전체 경력 기간 (선거 이력 조회 후 정치 입문 시점부터)
- 법적 고려사항:
  - **확정 판결만 포함**: "의혹", "수사", "기소" 단계는 배제
  - **1심·2심은 배제**: "대법원 확정", "상고기각", "판결 확정" 표현 확인 필수
  - **기사 보관**: 제목, 언론사, 날짜, URL만 저장 (전문 저장 금지)
- 신뢰성 검증:
  - 대법원 판결문 시스템에서 교차 확인 필수
  - 복수 언론사 보도 여부 확인
  - 법원 보도자료 확인 (중대 사건)
- 수집 난이도: 중상

**데이터 유형**: 공공+언론 병행 권장
**언론/SNS 필요성**: 중간

- 키워드:
  - "[의원명] + 재산 급증"
  - "[의원명] + 부정 축재"
  - "[의원명] + 재산신고 논란"
- 수집 출처: 정치 전문 언론, 시민단체 보도자료
- 수집 방법: 의원명 + "재산" 키워드 검색
- 법적 고려: 추정성 보도는 배제, 공식 조사 결과만 반영
- 활용 목적: 통계적 이상치 발견 시 배경 조사

## 데이터 신뢰성 검증 방법

### 1. 언론 데이터 신뢰성 검증

#### A. 언론사 등급 분류
**1급 (신뢰도 높음)**:
- 주요 중앙 일간지 (조선, 중앙, 동아, 한겨레, 경향)
- 법률 전문지 (법률신문, 법조신문)
- 공공기관 보도자료

**2급 (신뢰도 중간)**:
- 지역 일간지
- 온라인 전문 매체 (뉴스토마토, 미디어오늘)

**3급 (검증 필수)**:
- 개인 블로그
- 커뮤니티 게시물 (제외 권장)

#### B. 팩트체크 기준
1. **복수 출처 확인**: 2개 이상 언론사 보도
2. **공식 기록 대조**: 법원·선관위·감사원 기록 확인
3. **날짜 일치**: 사건 발생일과 보도일 일치 여부
4. **정정 보도 확인**: 후속 정정 보도 여부 확인

#### C. 신뢰도 점수화
```python
def calculate_reliability_score(article):
    score = 0

    # 언론사 등급
    if article['media_source'] in ['조선일보', '중앙일보', '한겨레']:
        score += 3
    elif article['media_source'] in ['지역일보']:
        score += 2

    # 복수 출처
    if article['multiple_sources'] >= 2:
        score += 2

    # 공식 기록 확인
    if article['official_record_verified']:
        score += 3

    # 확정 표현
    if '확정' in article['title'] or '판결' in article['title']:
        score += 2

    return score  # 최대 10점
```
