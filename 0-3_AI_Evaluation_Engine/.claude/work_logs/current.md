**수정 사항**:
1. 라인 14: 평가량 10,000개 → 4,000개 수정
2. 등급 체계: 8단계 → 9단계 (0 포함)

### 5. 혼동 주는 3,000개 언급 제거 (오늘 작업)

**사용자 피드백**: "왜 3000 개라는 말이 나온 거야 데이터 수집을 전개만 하면 되는데"
- 60+30+10=100은 명백한데 굳이 3,000 예시로 혼동시킬 필요 없음

**수정 내용**:
1. "실수 1: 수집량 계산 오류" 섹션 전체 제거 (라인 575-589)
2. "실수 2" → "실수 1"로 변경
3. "실수 3" → "실수 2"로 변경
4. 체크리스트 2곳에서 "(3,000개 아님!)" 제거
5. 핵심 기억 사항: ", 3,000개 아님!" → "(60% + 30% + 10% = 100%)"로 변경

**검증 결과**: 전체 통과 ✅
