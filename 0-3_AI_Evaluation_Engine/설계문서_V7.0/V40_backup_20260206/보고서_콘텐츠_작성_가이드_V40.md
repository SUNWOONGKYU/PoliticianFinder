# 보고서 콘텐츠 작성 가이드 V40

**목적**: AI 정치평론가가 유튜브 영상 제작 시 참고할 보고서의 콘텐츠 작성 방법

**대상**: 보고서 작성자 (AI/인간)

**버전**: V40.0

---

## 📌 핵심 원칙

### 보고서의 목적

- ✅ AI 정치평론가가 영상 제작 시 참고하는 **데이터 자료집**
- ✅ 풍부한 데이터와 명확한 근거 제공
- ❌ 스크립트가 아님 (AI 정치평론가가 자체 작성)
- ❌ 주관적 해석 최소화 (데이터 중심)

### 작성 3대 원칙

1. **객관성**: 편향 없는 중립적 서술
2. **근거 중심**: 모든 주장에 데이터/출처 명시
3. **구조화**: AI 정치평론가가 찾기 쉽게 정리

## 1. 톤 앤 매너 가이드

### 기본 원칙 3가지

**원칙 1: 중립적 사실 서술**
- 편향 없는 표현 사용
- "좋다/나쁘다" 대신 데이터 제시
- 확정된 사실만 단정적 표현

**원칙 2: 데이터 기반 서술**
- 모든 주장에 근거 명시
- 출처 표기 필수
- 통계와 수치 활용

**원칙 3: 논란은 객관적으로**
- 확정 vs 의혹 명확히 구분
- 양측 입장 균형있게 서술
- 진행 중 사안은 현재 상태 표기

---

### ✅ 좋은 표현 예시 (10개)

**1. 데이터 중심 표현**
```
✅ "ChatGPT 평가 881점으로 4개 AI 중 최고점"
✅ "조은희 의원은 leadership 카테고리에서 평균 대비 +12점 우위"
✅ "시민 평가 4.2/5.0점 (응답자 1,247명)"
```

**2. 중립적 논란 서술**
```
✅ "2022년 3월 재산 신고 누락 의혹으로 특검 조사 진행 중"
✅ "A 기업 후원금 수령 논란 (2023.05.12 기사)"
✅ "공약 이행률 65% (17개 중 11개 완료, 6개 진행 중)"
```

**3. 객관적 평가 인용**
```
✅ "Claude 평가 근거: '국정감사 우수의원 선정으로 전문성 입증'"
✅ "Gemini +3점 평가: '재난 대응 신속성 인정'"
✅ "AI 4개 모두 +2점 이상 (합의 평가)"
```

**4. 확정 vs 의혹 구분**
```
✅ "2021년 정치자금법 위반으로 벌금 500만 원 확정 (대법원 판결)"
✅ "이해충돌 의혹 제기됨 (검찰 수사 중, 결론 미확정)"
```

**5. 통계 표현**
```
✅ "AI 평가 표준편차 8.9점 (평가 일관성 높음)"
✅ "긍정 평가 75.0% (ChatGPT), 65.4% (Claude)"
✅ "상위 15% 점수 (전체 정치인 평균 750점 대비 +66점)"
```

**6. 비교 표현**
```
✅ "타 정치인 평균 대비 leadership +12점, integrity -5점"
✅ "같은 정당 내 순위 3위 (총 18명 중)"
✅ "전국 기초단체장 평균 782점 대비 +34점"
```

**7. 시간 범위 명시**
```
✅ "2016년 이후 활동 기준 (최근 8년)"
✅ "2023년 7월~2024년 6월 데이터"
✅ "현재 진행 중 (2024년 9월 기준)"
```

**8. 긍정 사례**
```
✅ "2023년 6월 스마트시티 조례 제정으로 시민 편의 증대 (ChatGPT +3)"
✅ "재난 안전망 구축으로 응급 대응 시간 30% 단축 (Grok +4)"
```

**9. 부정 사례**
```
✅ "2022년 예산 낭비 지적 (감사원 보고서, Gemini -2)"
✅ "공약 지연으로 시민 불만 제기 (Claude -3)"
```

**10. AI 평가 종합**
```
✅ "AI 4개 중 3개가 +3 이상 (ChatGPT +3, Grok +3, Gemini +3, Claude +2)"
✅ "평가 편차 1.27점 (일관성 매우 높음)"
```

### ❌ 나쁜 표현 예시 (10개)

**1. 편향적 표현**
```
❌ "훌륭한 정치인"
❌ "최악의 선택"
```

```
❌ "성공적인 정책" (평가 기준 없음)
```

**3. 확정되지 않은 추측**
```
❌ "아마도 부정부패일 것"
❌ "~인 것으로 보임" (진행 중 사안에 단정)
❌ "의혹이 사실이라면" (가정 서술)
```

**4. 과장 표현**
```
❌ "엄청난 성과"
❌ "완벽한 리더십"
❌ "심각한 문제"
```

**5. 애매모호한 표현**
```
❌ "어느 정도 좋음"
❌ "상당히 높은 점수"
❌ "대체로 긍정적"
```

**6. 인신공격**
```
❌ "능력 없는 정치인"
❌ "무책임한 태도"
❌ "국민을 무시함"
```

**7. 불명확한 시간**
```
❌ "최근에"
❌ "오래전"
❌ "한때"
```

**8. 출처 누락**
```
❌ "재산을 숨겼다고 함" (출처 없음)
❌ "스캔들이 있었음" (구체성 없음)
```

**9. 주관적 해석**
```
❌ "이 점수는 낮다고 볼 수 있음"
❌ "AI가 잘못 평가한 것 같음"
❌ "실제로는 더 좋을 것"
```

**10. 비교 기준 누락**
```
❌ "높은 점수를 받았음" (비교 대상 없음)
❌ "우수한 성과" (기준 없음)
```

### 중립적 서술 가이드라인

**논란 서술 시 4단계 체크리스트:**

- [ ] **확정 사실인가, 의혹인가?**
  - 확정: "~로 확정" (판결, 조사 결과)
  - 의혹: "~의혹 제기" (수사 중, 미확정)

- [ ] **출처가 명확한가?**
  - 법원 판결문, 감사원 보고서, 언론 보도 등

- [ ] **시간 정보가 정확한가?**
  - 날짜, 기간 명시

- [ ] **양측 입장이 있는가?**
  - 논란 사안은 양측 입장 모두 서술

**예시:**
```
✅ "2022년 3월 재산 신고 누락 의혹 제기 (A일보 보도)
    - 의혹: 재산 5억 원 미신고
    - 본인 해명: '배우자 명의로 신고 완료'
    - 현재 상태: 선관위 조사 진행 중 (2024.09 기준)
    - AI 평가: Claude -3, Gemini -2 (윤리성 카테고리)"
```

## 2. 데이터 표현 가이드

### 점수 표현 방법

**기본 형식:**
```
[점수] ([비교 대상] [차이])
```

**예시 (5가지):**

**1. 절대 점수 + 등급**
```
✅ "816점 (E등급 - Emerald)"
✅ "881점 (D등급 - Diamond)"
```

**2. 상대 비교 (평균 대비)**
```
✅ "816점 (전체 평균 750점 대비 +66점)"
✅ "738점 (Claude 평균 780점 대비 -42점)"
```

**3. 백분위**
```
✅ "816점 (상위 15%)"
✅ "881점 (상위 5%)"
```

**4. AI별 점수**
```
✅ "ChatGPT 881점 / Grok 835점 / Gemini 807점 / Claude 738점 (평균 816점)"
```

**5. 카테고리별 점수**
```
✅ "Leadership 87점 (상위 10%) / Integrity 72점 (평균 수준)"
```

---

### 통계 표현 방법

**표준편차 설명:**
```
❌ "표준편차 8.9점"
✅ "AI 평가 편차 8.9점 (일관성 매우 높음)"
✅ "AI 간 평가 차이 작음 (표준편차 8.9점)"
```

**백분율 설명:**
```
❌ "긍정 평가 75.0%"
✅ "ChatGPT 긍정 평가 75.0% (+3/+4 비율)"
✅ "4개 AI 중 3개(75%)가 +3 이상 평가"
```

**비율 비교:**
```
✅ "ChatGPT 긍정 비율 75.0% vs Claude 65.4% (차이 9.6%p)"
✅ "X 판정 비율: Gemini 16.6% (최저) / Claude 29.3% (최고)"
```

**평균 등급 설명:**
```
❌ "평균 등급 +2.77"
✅ "ChatGPT 평균 등급 +2.77 (거의 +3에 가까움)"
✅ "Claude 평균 등급 +1.50 (가장 보수적 평가)"
```

---

### 비교 표현 방법

**1. 타 정치인 대비**
```
✅ "Leadership 87점 (정치인 평균 75점 대비 +12점)"
✅ "Integrity 68점 (정치인 평균 73점 대비 -5점)"
```

**2. 같은 정당 내**
```
✅ "같은 정당 18명 중 3위"
✅ "당내 평균 780점 대비 +36점"
```

**3. 같은 직급 내**
```
✅ "기초단체장 평균 782점 대비 +34점"
✅ "전국 구청장 120명 중 상위 15%"
```

**4. AI 간 비교**
```
✅ "ChatGPT 최고점 (881점) vs Claude 최저점 (738점), 차이 143점"
✅ "4개 AI 중 3개가 800점 이상"
```

**5. 시간별 비교**
```
✅ "2023년 평가 782점 → 2024년 816점 (+34점 상승)"
```

## 3. 사례 서술 템플릿

### 긍정 사례 템플릿

**구조:**
```
[제목] ([날짜])
- 내용: [무엇을 했는가]
- 효과: [시민에게 어떤 도움이 되었는가]
- AI 평가: [어떤 AI가 어떻게 평가했는가]
- 출처: [링크 또는 참조]
```

**실제 예시 1:**
```
**스마트시티 조례 제정** (2023년 6월)
- 내용: IoT 기반 도시 안전망 구축 조례 제정
- 효과: 범죄 예방 20% 향상, 응급 대응 시간 30% 단축
- AI 평가: ChatGPT +3 ("혁신적 정책"), Grok +4 ("시민 안전 크게 개선")
- 출처: 시의회 의안 제2023-45호
```

```
**재난 대응 체계 구축** (2023년 8월)

- 효과: 재난 발생 시 골든타임 확보, 피해 최소화
```

**실제 예시 3:**
```
**주민 소통 플랫폼 개설** (2023년 3월)
```

## 4. 섹션별 작성 가이드

### 섹션 1: 정치인 프로필

**목적:**
- 정치인의 기본 정보 제공
- AI 정치평론가가 영상 도입부에서 사용

**톤:**
- 중립적, 사실 중심
- 긍정/부정 평가 배제
- 나열식 서술

**구조:**
```
1. 기본 정보
   - 이름, 나이, 정당
   - 현재 직책

2. 학력 및 경력
   - 시간순 나열 (최신 → 과거)
   - 주요 이력만 선택

3. 주요 활동 분야
   - 전문 분야 3개 이내
```

**필수 요소:**
- 이름
- 현재 직책
- 정당
- 주요 경력 3개

**선택 요소:**
- 출생지
- 학력
- 가족 사항
- 취미/특기

**예시 문장 3개:**

## 5-1. ChatGPT 인용 규칙

### 기본 원칙

ChatGPT가 제시한 예측이나 추측을 보고서에 포함할 때는 **반드시 ChatGPT 출처를 명확히** 해야 합니다.

### ❌ 잘못된 예시 (ChatGPT 추측을 보고서 주장처럼 표현)

```
❌ "이러한 활동을 통해 향후 시민 만족도가 더욱 높아질 것으로 기대됩니다."
   → 누가 기대하는가? 보고서 작성자? ChatGPT? 불명확!

❌ "리더십이 계속 발휘될 것입니다."
   → 미래 예측의 근거가 없음

❌ "앞으로도 청렴도가 유지될 전망입니다."
   → 누가 전망하는가? 출처 불명
```

### ✅ 올바른 예시 (ChatGPT 출처 명확히)

```
✅ "ChatGPT는 '이러한 활동을 통해 향후 시민 만족도가 더욱 높아질 것'으로 예측했습니다."
   → ChatGPT가 예측했다고 명확히 표기

✅ "ChatGPT 평가 근거: '리더십이 계속 발휘될 것으로 기대됨'"
   → 평가 근거 인용문으로 표시

✅ "ChatGPT는 '앞으로도 청렴도가 유지될 것'으로 평가"
   → AI 평가로 명확히 구분
```

### 인용 형식

**형식 1: 직접 인용**
```
ChatGPT는 "[원문 그대로]"라고 평가했습니다.
```

**형식 2: 평가 근거 인용**
```
ChatGPT 평가 근거: "[원문 그대로]"
```

**형식 3: 예측 표기**
```
ChatGPT는 ~할 것으로 예측/전망/기대
```

### 중요 체크포인트

- [ ] 미래 예측 표현이 있는가?
- [ ] 그것이 ChatGPT 평가에서 온 것인가?
- [ ] ChatGPT 출처를 명확히 표기했는가?
- [ ] 보고서 자체 주장과 구분되는가?

### 예외: 과거 사실 기반 표현

```
✅ "2023년 스마트시티 조례 제정 이후 범죄율이 20% 감소했습니다."
   → 과거 사실, ChatGPT 인용 불필요

✅ "2024년 9월 기준, 공약 이행률 65%를 기록하고 있습니다."
   → 현재 사실, ChatGPT 인용 불필요

❌ "이러한 추세가 계속될 것입니다."
   → 미래 예측, ChatGPT 출처 필요
```

## 7. 최종 점검 항목

### 필수 확인 사항

**1. 객관성**
- 편향 없는 중립적 서술
- 확정 사실만 단정적 표현
- 의혹은 "의혹 제기" 표기

**2. 근거 명확성**
- 모든 주장에 데이터/출처
- 통계 정확성
- 출처 링크 작동

**3. 균형**
- 긍정/부정 사례 균형
- 논란 사안 양측 입장
- AI 평가 차이 설명

**4. 이해 용이성**
- 간결한 문장
- 명확한 표현
- 구조화된 정리

**5. AI 정치평론가 활용성**
- 참고하기 쉬운 구조
- 풍부한 데이터
- 인용 가능한 문장
