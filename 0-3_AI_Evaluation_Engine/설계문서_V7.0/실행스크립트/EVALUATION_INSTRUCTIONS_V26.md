**버전**: V26.0 Final

## 1. 평가 개요

### 풀링 방식
- 4개 AI (Claude, ChatGPT, Grok, Gemini)가 수집한 데이터 풀
- 카테고리당 200개 풀 (4 AI × 50개)
- 각 AI가 200개 전체를 독립적으로 평가

### 핵심 원칙
- **타이밍 분리**: 수집과 평가는 별도 세션
- **독립적 판단**: 다른 AI가 수집한 데이터도 객관적으로 평가
- **정치인 맥락 유지**: 누구를 평가하는지 반드시 인지

### ⚠️ 등급 선택 규칙 (CRITICAL!)

#### 🚫 절대 금지
- ❌ B+, B-, A+, A- (플러스/마이너스 기호 금지)

| 등급 | 점수 | 의미 | 설명 |
|------|------|------|------|
| **A** | +8 | 최고 수준 | 역사적 업적, 세계적 수준 |
| **B** | +6 | 매우 우수 | 명확한 탁월성, 국내 최고 |
| **C** | +4 | 우수함 | 일반적 긍정, 평균 이상 |

| 등급 | 점수 | 의미 | 설명 |
|------|------|------|------|
| **E** | -2 | 미흡 | 약한 부정, 개선 필요 |
| **F** | -4 | 부족 | 명확한 문제점 존재 |
| **G** | -6 | 매우 부족 | 심각한 문제, 실패 |
| **H** | -8 | 심각 | 최악, 위법/부패 |

## 4. 등급별 상세 기준

### A 등급 (+8점) - 최고 수준
- 해당 분야에서 역사에 남을 만한 탁월한 성과
- 세계적 수준의 전문성 또는 업적
- 국제적으로 인정받는 혁신적 정책
- 명백하고 측정 가능한 탁월한 성과

**예시**:
- 전문성: 해외 명문대 박사 + 20년 경력 + 국제학회 수상
- 청렴성: 재산 공개 모범 사례, 부패 척결 성과

### B 등급 (+6점) - 매우 우수
- 국내 최고 수준의 전문성 또는 성과
- 명확하게 우수한 업적이 다수 존재
- 해당 분야에서 일반적으로 인정받는 성공 사례
- 논란 없이 명백한 긍정적 평가

**예시**:
- 전문성: 국내 명문대 박사 + 15년 경력 + 다수 논문
- 리더십: 조직 혁신 성공, 위기 대응 모범 사례

### C 등급 (+4점) - 우수함
- 평균 이상의 긍정적 평가
- 일부 긍정적 성과 존재
- 큰 문제는 없으나 탁월하지는 않음
- 일반적으로 긍정적으로 평가됨

**예시**:
- 전문성: 석사학위 + 10년 경력 + 일부 전문 활동
- 소통: SNS 활용 양호, 시민 대화 노력

### D 등급 (+2점) - 보통
- 약한 긍정 또는 중립에 가까운 평가
- 최소한의 긍정적 요소 존재
- 특별한 성과는 없으나 기본은 갖춤
- 평균 수준

**예시**:
- 전문성: 학사학위 + 5년 경력
- 책임성: 공약 일부 이행, 기본 의무 수행

### E 등급 (-2점) - 미흡
- 약한 부정 또는 중립에 가까운 평가
- 일부 미흡한 점 존재
- 큰 문제는 아니나 개선 필요
- 기대에 미달

**예시**:
- 전문성: 경력 3년 미만, 전문성 부족 지적
- 대응성: 민원 처리 지연, 현장 방문 부족

### F 등급 (-4점) - 부족
- 평균 이하의 부정적 평가
- 명확한 문제점 존재
- 여러 미흡한 사례 확인됨
- 비판이 지속됨

**예시**:
- 청렴성: 이해충돌 논란, 정치자금 문제
- 책임성: 공약 다수 파기, 책임 회피

### G 등급 (-6점) - 매우 부족
- 명확하고 심각한 문제 존재
- 부정적 평가가 지배적
- 실패 사례가 명백함
- 신뢰 상실

**예시**:
- 청렴성: 비리 의혹 수사 중, 기소
- 윤리성: 반복적 막말, 품위 손상

### H 등급 (-8점) - 심각
- 역사적으로 심각한 실패 또는 위법 행위
- 회복 불가능한 수준의 문제
- 명백한 부패, 범죄, 또는 중대한 과실
- 유죄 판결, 구속

**예시**:
- 청렴성: 뇌물 수수 유죄 확정, 횡령 판결
- 전문성: 학력 위조 확정 판결

## 5. 10개 카테고리별 평가 기준

### Category 1: Expertise (전문성)

**【영문 정의】** The level of knowledge, skills, and experience required to perform duties effectively

**【한글 정의】** 정책 전문성, 행정 경험, 분야별 전문 지식

**[O] 전문성에 해당:** 정책 수립 능력, 전문 지식, 학력/경력, 행정 경험, 법안 발의
**[X] 전문성 아님:** 조직 관리(→리더십), 혁신(→비전), 약속 이행(→책임성)

**측정 지표:**
- 1순위: 최고 학력 (A: 해외 명문대 박사 ~ H: 학력 위조 확정)
- 2순위: 경력 기간 (A: 20년+ 고위직 ~ H: 경력 위조 확정)
- 3순위: 전문 자격증 (A: 변호사+박사 ~ H: 자격증 위조)

---

### Category 2: Leadership (리더십)

**【영문 정의】** The ability to effectively lead organizations and people to achieve goals

**【한글 정의】** 조직 관리 능력, 위기 대응, 의사결정 능력

**[O] 리더십에 해당:** 조직 관리, 팀 구성, 위기 대응, 갈등 조정, 의사결정
**[X] 리더십 아님:** 정책 전문성(→전문성), 장기 계획(→비전), 시민 소통(→소통)

**측정 지표:**
- 조직 성과, 위기 대응 사례, 의사결정 평가, 당내 영향력

### Category 3: Vision (비전)

**【영문 정의】** The ability to predict the future and present long-term goals

**【한글 정의】** 장기적 계획, 혁신성, 미래 전망

**[O] 비전에 해당:** 장기 발전 계획, 미래 전략, 혁신 정책, 창의성, 개혁 의지
**[X] 비전 아님:** 현재 정책(→전문성), 조직 혁신(→리더십), 단기 공약(→책임성)

**측정 지표:**
- 장기 계획 수립, 혁신 정책 추진, 미래 전략 제시

### Category 4: Integrity (청렴성)

**【영문 정의】** The quality of not engaging in financial or material corruption

**【한글 정의】** 부패 방지, 윤리 준수, 이해충돌 회피

**[O] 청렴성에 해당:** 금품 수수, 횡령, 배임, 비리, 이해충돌, 불법 정치자금, 직권 남용
**[X] 청렴성 아님:** 정치적 중립성(→책임성), 도덕성(→윤리성), 정보 공개(→투명성)

**측정 지표:**
- 재산 공개, 정치자금 투명성, 감사 결과, 법적 판결

**【한글 정의】** 도덕성, 사회적 책임, 공정성

**[O] 윤리성에 해당:** 도덕성, 가치관, 인격, 공정성, 차별 금지, 인권 존중, 품위
**[X] 윤리성 아님:** 금품 수수(→청렴성), 공약 불이행(→책임성), 정보 은폐(→투명성)

## 6. 등급 선택 프로세스

### 4단계 프로세스
1. **내용 분석** → 긍정인가 부정인가 판단
2. **강도 평가** → 약한가 강한가 판단
3. **등급 선택** → A, B, C, D, E, F, G, H 중 1개 선택
4. **검증** → 선택한 등급이 8개 중 하나인지 확인

### 등급 선택 가이드
```
긍정적 내용:
├── 매우 강함 → A
├── 강함 → B
├── 보통 → C
└── 약함 → D

부정적 내용:
├── 약함 → E
├── 보통 → F
├── 강함 → G
└── 매우 강함 (위법/범죄) → H
```

```json
{
  "evaluations": [
    {
      "item_index": 1,
      "rating": "B",
      "rating_rationale": "평가 근거 (50-150자)"
    },
    {
      "item_index": 2,
      "rating": "F",

    }
  ]

```

### 필드 설명
| 필드 | 설명 | 필수 |
|------|------|------|
| item_index | 평가 대상 항목 번호 | ✅ |
| rating | A~H 중 하나 | ✅ |
| rating_rationale | 등급 선택 근거 (50-150자) | ✅ |

### ⚠️ 주의사항
- rating은 반드시 A, B, C, D, E, F, G, H 중 하나
- rating_rationale은 왜 이 등급인지 명확히 설명
- item_index는 제공된 항목 번호와 일치

## 8. 평가 시 주의사항

### 함정 (Pitfalls)

1. **정치인 맥락 무시 금지**: 반드시 정치인 정보를 고려하여 평가
2. **학벌 과대평가**: 명문대 출신이라도 경력이 짧으면 C~D 등급
3. **경력 허수 구별**: 명목상 경력 vs. 실질적 경력 구분
4. **시대 고려**: 과거 교육 환경 고려
5. **확정 판결 기준**: 유죄 확정 시에만 H 등급

### 객관성 유지 원칙

1. **정치인 인지**: 누구를 평가하는지 반드시 인지
2. **데이터 기반**: 제공된 데이터 내용을 기반으로 평가
3. **카테고리 집중**: 해당 카테고리에 맞는 평가만
4. **일관성 유지**: 같은 수준의 내용에 같은 등급
5. **극단 등급 신중**: A와 H는 명백한 경우에만

## 9. 체크리스트

평가 완료 전 확인:
- [ ] 정치인 프로필 정보 확인
- [ ] 카테고리 정의 확인
- [ ] rating: A~H 중 하나만 선택
- [ ] rating_rationale: 등급 선택 근거 명확히 작성
- [ ] item_index: 모든 항목에 대해 평가 완료
- [ ] 극단 등급 (A, H) 사용 시 근거 충분한지 확인

## 10. 올바른 예시 vs 잘못된 예시

### ✅ 올바른 예시
```json
{
  "evaluations": [
    {
      "item_index": 1,
      "rating": "B",
      "rating_rationale": "오세훈 시장의 도시계획 전문성이 명확히 드러나는 사례. 15년 이상의 행정 경험과 다수의 정책 수립 실적이 확인됨."
    },
    {
      "item_index": 2,
      "rating": "F",
      "rating_rationale": "재개발 정책 추진 과정에서 주민 의견 수렴 부족이 명확히 지적됨. 여러 민원이 제기된 사례."
    }
  ]
}
```

### ❌ 잘못된 예시
```json
{
  "evaluations": [
    {
      "item_index": 1,
      "rating": "B+",  // ❌ B+는 금지! B만 사용
      "rating_rationale": "좋음"  // ❌ 너무 짧음
    },
    {
      "item_index": 2,
      "rating": "5",  // ❌ 숫자 금지! A~H만
      "rating_rationale": "..."
    }
  ]
}
```
