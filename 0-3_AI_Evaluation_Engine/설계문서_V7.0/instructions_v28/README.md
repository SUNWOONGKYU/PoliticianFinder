## 6. 점수 계산

### 등급 → 점수 변환

```
등급 × 2 = 점수

+4 × 2 = +8점
+3 × 2 = +6점
+2 × 2 = +4점
+1 × 2 = +2점
-1 × 2 = -2점
-2 × 2 = -4점
-3 × 2 = -6점
-4 × 2 = -8점
```

### 카테고리 점수 (20~100점)

```python
PRIOR = 6.0
COEFFICIENT = 0.5
category_score = (PRIOR + avg_rating * COEFFICIENT) * 10
```

### 최종 점수 (200~1000점)

```python
total_score = sum(10개 카테고리 점수)
# 범위: 200 ~ 1000점
```
