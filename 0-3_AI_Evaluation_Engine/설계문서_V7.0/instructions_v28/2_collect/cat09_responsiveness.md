# Category 09: Responsiveness (대응성) - 수집 지침서 V28

---

## 1. 카테고리 정의

### 한글 정의
시민의 요구와 민원에 신속하게 대응하는 능력

### 영문 정의
The ability to respond promptly to citizens' demands and complaints

### 학술 근거
- **Vigoda (2002)**: 행정 대응성의 구성 요소
- **Putnam (1993)**: 민주주의와 시민 대응성
- **Bryer (2007)**: 관료제의 대응성 이론

## 8. 주의사항 (Pitfalls)

1. **권한 범위 확인**: 해당 정치인 권한 내 사안인지 확인
2. **응답 vs 해결 구분**: 답변만 한 것과 실제 해결한 것 구분
3. **우선순위 맥락**: 민원 긴급성/중요성 맥락 기록
4. **제도적 한계**: 법적/제도적 한계로 대응 불가한 경우 구분
5. **시스템 vs 개인**: 시스템 문제 vs 개인 대응 문제 구분

## 8-1. 중복 수집 금지 (CRITICAL!)

### 핵심 규칙
**동일한 사건/주제는 반드시 1개만 수집!**

### 금지 사항
- 같은 민원 대응 사례를 반복 수집 ❌
- 같은 현장 방문을 다른 출처로 반복 ❌

### 예시: 중복 수집 (잘못된 예)
```
[X] "재난 현장 방문" (출처 A)
[X] "수해 현장 점검" (출처 B)
[X] "침수 지역 방문" (출처 C)
→ 같은 방문이면 1개만!
```

**중복 수집 시 전체 응답 무효 처리됨!**
