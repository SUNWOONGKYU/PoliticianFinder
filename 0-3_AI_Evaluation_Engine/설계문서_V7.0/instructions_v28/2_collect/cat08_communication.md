# Category 08: Communication (소통능력) - 수집 지침서 V28.0

---

## 1. 카테고리 정의

### 한글 정의
시민과 효과적으로 의사소통하는 능력

### 영문 정의
The ability to communicate effectively with citizens

### 학술 근거
- **Habermas (1984)**: 의사소통적 합리성 이론
- **Fishkin (2009)**: 숙의 민주주의와 소통
- **Nabatchi (2012)**: 시민 참여와 소통의 관계

## 8. 주의사항 (Pitfalls)

1. **일방 vs 쌍방 구분**: 일방적 홍보 vs 실질적 소통 구분
2. **양 vs 질 구분**: SNS 게시물 수 ≠ 소통 품질
3. **선별 소통 주의**: 특정 집단만 소통하는지 확인
4. **대리 소통 구분**: 본인 소통 vs 보좌관 대리 소통
5. **시기 고려**: 선거 기간 vs 평상시 소통 패턴

## 8-1. 중복 수집 금지 (CRITICAL!)

### 핵심 규칙
**동일한 사건/주제는 반드시 1개만 수집!**

### 금지 사항
- 같은 소통 행사를 반복 수집 ❌
- 같은 SNS 활동을 다른 출처로 반복 ❌

### 예시: 중복 수집 (잘못된 예)
```
[X] "타운홀 미팅 개최" (출처 A)
[X] "시민 간담회 진행" (출처 B)
[X] "주민 대화 행사" (출처 C)
→ 같은 행사면 1개만!
```

**중복 수집 시 전체 응답 무효 처리됨!**
