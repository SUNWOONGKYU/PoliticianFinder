# Category 07: Transparency (투명성) - 평가 지침서 V40

---

## 1. 카테고리 정의

### 한글 정의
정보를 숨김없이 공개하고 공유하는 자질

### 영문 정의
The quality of openly disclosing and sharing information without concealment

### 학술 근거
- **Florini (2007)**: 투명성의 민주적 가치
- **Hood (2006)**: 정부 투명성 측정 지표
- **Grimmelikhuijsen (2012)**: 투명성과 신뢰의 관계


## 2. V40 평가 방식 - 풀링 평가

### 평가 대상 데이터

수집 단계에서 collected_data_v40 테이블에 저장된 풀링 데이터 전체를 평가합니다.

4개 평가 AI (Claude, ChatGPT, Gemini, Grok)가
각각 전체 풀링 데이터를 독립적으로 평가합니다.

### 평가 독립성

```
타이밍 분리 = 객관성 보장

[평가 단계] - 수집과 다른 세션, 다른 시점
→ Claude, ChatGPT, Gemini, Grok이 평가
→ 수집 시점 ≠ 평가 시점
→ 독립적 판단 = 더 객관적
```

## 4. 등급 체계 (+4 ~ -4) - V40

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 투명성의 모범, 적극적 정보 공개 주도 |
| **+3** | +6 | 우수 - 정보 공개에 적극적, 재정 투명 공개 |
| **+2** | +4 | 양호 - 투명하게 활동, 정보공개 요청 응답 |
| **+1** | +2 | 경미한 긍정 - 기본적인 투명성 유지 |

| **-1** | -2 | 경미한 부정 - 일부 정보 공개 소극적 |
| **-2** | -4 | 일반 부정 - 투명성 문제 제기, 정보공개 거부 |
| **-3** | -6 | 심각한 부정 - 정보 은폐 경향, 자료 제출 거부 |
| **-4** | -8 | 최악 - 심각한 불투명, 기록 삭제/조작 |

| **X** | 0 | 평가 제외 - 10년 이상 과거, 동명이인, 무관 내용, 날조 정보 |

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 투명성의 모범 사례
- 적극적 정보 공개 주도
- 재정 완전 투명 공개
- 시민단체 투명성 평가 최우수

### +3등급 (우수함)
- 정보 공개에 적극적
- 정책 결정 과정 설명
- 재정 투명하게 운영
- 언론 질의에 성실 대응

### +2등급 (양호함)
- 전반적으로 투명하게 활동
- 정보 공개 요청에 응답
- 기본적인 투명성 유지
- 큰 문제 없음

### +1등급 (경미한 긍정)
- 기본적인 투명성
- 특별한 문제 없음
- 평균 수준
- 무난한 평가

### -1등급 (경미한 부정)
- 일부 정보 공개 소극적
- 비공개 결정 사례 있음
- 투명성 개선 필요
- 약간의 비판

### -2등급 (일반 부정)
- 투명성 문제 제기됨
- 정보 공개 거부 사례
- 시민단체 비판
- 불투명 경향

### -3등급 (심각한 부정)
- 정보 은폐 경향
- 다수의 비공개 결정
- 자료 제출 거부
- 심각한 불투명

### -4등급 (최악)
- 심각한 불투명
- 조직적 정보 은폐
- 기록 삭제/조작
- 정치인 부적합

## 6. 평가 프로세스

### 입력 데이터
```
해당 카테고리의 풀링 데이터 전체
(collected_data_v40 테이블에서 조회)
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 채널과 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **법적 비공개**: 법적으로 비공개 가능한 정보 구분
2. **공개 범위**: 부분 공개 vs 전체 공개
3. **적시성**: 정보 공개 시점 (요청 전/후)
4. **형식적 공개**: 형식만 갖춘 공개 vs 실질적 공개
