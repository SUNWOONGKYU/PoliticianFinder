**핵심 포인트**:
- **PVI(Partisan Voting Index) 자체 계산** 필요 (한국에 공식 지표 없음):
  ```
  한국 PVI = (해당 선거구 여당 득표율 - 전국 여당 평균 득표율)
  최근 2회 선거 결과 평균 사용
  ```
- 데이터: 선관위 개방포털 + 공공데이터포털 API로 역대 선거구별 투개표 결과 조회
- 정당 프리미엄: 영호남 +15~30%p [강원택(2016)]
- 정당 지지율: 갤럽/리얼미터 주간 조사
