# Alpha 2-3: Regional (지역기반) - 평가 지침서 V60 CCI

---

## 1. 카테고리 정의

### 한글 정의
출생지·학력·경력 연고, 지역 조직 동원력, 지역구 서비스 활동을 종합한 지역 기반 경쟁력

### 영문 정의
Regional competitive advantage from hometown ties, organizational mobilization, and constituency service activities

### 학술 근거
- **이갑윤 (2011)**: 지역 연고와 득표율 — 출생지·학력 연고 +3~20%p
- **Fenno (1978)**: Home Style — 지역구 활동이 활발한 의원 재선율 +10~15%p
- **박찬욱 (2000)**: 한국 지역주의와 선거 행태

## 2. 평가 방식 — 상대평가 (플래툰 포메이션)


## 4. 등급 체계 (+4 ~ -4)

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **토착 후보**: 출생+학력+경력 3중 연고, 강력한 조직, 대규모 후원금 |
| +3 | +6 | **강한 지역기반**: 2중 이상 연고, 조직 탄탄, 지역 인사 지지 |
| +2 | +4 | **유리한 지역기반**: 1개 이상 연고, 적절한 조직, 지역 서비스 |
| +1 | +2 | **보통**: 지역 거주하나 연고 약함, 기본 조직 |
| -1 | -2 | **약한 기반**: 연고 없이 거주, 조직 미비, "낙하산" 인식 |
| -2 | -4 | **외부인**: 지역 연고 없음, 조직 부재, 후원금 적음 |
| -3 | -6 | **강한 외부인**: 타지역 출신 + 거주 이력 없음 + "카펫배거" |
| -4 | -8 | **지역 무관**: 지역 연결고리 전무, 기반 없음 |
| X | 0 | **제외**: 지역기반과 무관한 자료 |

## 5. 연고 판단 기준

```
3중 연고 (출생+학력+경력): +4 기반
  예: 해당 지역 출생 + 지역 고교 + 지역 활동 20년

2중 연고 (출생+학력 or 출생+경력): +3 기반
  예: 해당 지역 출생 + 타지 대학 + 지역 시의원 10년

1중 연고 (경력만 or 거주만): +1~+2 기반
  예: 타지 출신 + 해당 지역 국회의원 3선

연고 없음: -1~-2 기반
  예: 중앙 정치인이 지역구 변경

카펫배거 (완전 외부인): -3~-4
  예: 서울 출신이 타지역 출마 (거주 이력도 없음)
```

---

## 6. 판단 원칙

1. **연고의 깊이**: 출생 > 학력 > 경력 > 거주 순으로 강력
2. **조직력 반영**: 연고가 있어도 조직이 없으면 효과 감소
3. **지역 서비스**: 현직자의 지역구 예산·민원 해결 실적 반영
4. **경쟁자 비교**: 같은 그룹 내 후보별 지역 밀착도 비교
5. **rationale 한국어 1문장**
