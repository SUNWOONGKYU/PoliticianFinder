## 1. 카테고리 정의

### 한글 정의
미디어 보도 논조, SNS 버즈, 검색 관심도, 브랜드 메시지 일관성을 종합한 후보 이미지

### 영문 정의
Candidate image combining media tone, social media buzz, search interest, and brand message consistency

### 학술 근거
- **Prior (2006)**: 미디어 노출과 정치 지식·투표 — 긍정 보도 상위 25% → 지지율 +5~8%p
- **한규섭**: 정치 커뮤니케이션 — 프레이밍 효과
- **McCombs & Shaw (1972)**: 의제설정이론 — 미디어가 의제를 설정

|---|--------|------|------------|

### ❌ 이미지·내러티브 아님

```
❌ 여론조사 수치 → opinion (여론동향)

❌ 정당 이미지 → party (정당경쟁력)
❌ 의정활동 실적 → GPI 카테고리
```

## 4. 등급 체계 (+4 ~ -4)

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **압도적 긍정 이미지**: 미디어 노출 1위, 긍정 보도 70%+, SNS 호감 |
| +3 | +6 | **강력한 긍정**: 보도 논조 긍정 우세, 브랜드 일관성 높음 |
| +2 | +4 | **양호한 이미지**: 보도량 상위, 긍정/중립 우세 |
| +1 | +2 | **기본 노출**: 보도 있으나 두드러지지 않음, 브랜드 보통 |
| -1 | -2 | **부정 노출 시작**: 부정 보도 비율 증가, SNS 비판 |
| -2 | -4 | **부정 이미지 고착**: 부정 보도 우세, 브랜드 혼란 |
| -3 | -6 | **이미지 위기**: 대규모 부정 보도, 핵심 메시지 붕괴 |
| -4 | -8 | **이미지 파탄**: 미디어에서 부정적 상징으로 고착 |
| X | 0 | **제외**: 이미지·내러티브와 무관한 자료 |

## 5. X 판정 기준

**반드시 X:**
- 동명이인 관련 보도
- 해당 후보와 전혀 무관한 미디어 데이터
- 5년 이상 과거 이미지 데이터

**X가 아닌 낮은 등급:**
- 부정적 이미지라도 해당 후보에 대한 것 → -1 ~ -4
- 간접 언급 → +1

## 6. 판단 원칙

1. **감성분석 기반**: 빅카인즈/썸트렌드 감성분석 결과를 1차 기준으로 활용
2. **경쟁자 대비 비교**: 같은 그룹 후보의 미디어 점유율·논조 비교
3. **추이 중시**: 단일 시점보다 2~4주 트렌드 판단
4. **노출량 × 논조**: 노출이 많아도 부정이면 마이너스, 적어도 긍정이면 플러스
5. **rationale 한국어 1문장**
