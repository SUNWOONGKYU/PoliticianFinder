# Alpha 1-3: Risk (리스크) - 평가 지침서 V60 CCI

---

## 1. 카테고리 정의

### 한글 정의
네거티브 공격에 대한 취약점을 역산하여 평가하는 리스크 지수

### 영문 정의
Risk index measuring vulnerability to negative attacks through reverse-engineering potential attack vectors

### 학술 근거
- **이재묵 (2018)**: 네거티브 캠페인 효과(한국) — 효과적 네거티브 → 상대 후보 -3~7%p
- **Geer (2006)**: 네거티브 광고는 정보량이 많고 유권자 학습 효과
- **Lau & Rovner (2009)**: 네거티브 캠페인의 동원/탈동원 효과

---

## 2. 평가 방식 — 상대평가 (플래툰 포메이션)

- Claude 단독 평가 (소대장 조율)
- 같은 경쟁자 그룹 내 비교
- **역산 평가**: 리스크가 적을수록 높은 등급 (+4), 많을수록 낮은 등급 (-4)

### ✅ 리스크 평가 기준

| # | 항목명 | 설명 | 데이터 소스 |
|---|--------|------|------------|
| 3-1 | **부정 기사 비율** | 전체 보도 중 부정 기사 비율 | 빅카인즈 감성분석 |
| 3-2 | **법적 리스크** | 수사/기소/재판 이력, 전과 | 대법원 판결 검색 |

| 3-4 | **시민단체 지적** | 경실련/참여연대 낙천/낙선 명단 포함 | 경실련, 참여연대 |

❌ 정당 리스크 → party (정당경쟁력)

## 4. 등급 체계 (+4 ~ -4) — 역산 방식

⚠️ **리스크는 역산**: 리스크 없음 = +4, 리스크 심각 = -4

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **클린**: 부정 기사 5% 미만, 법적·재산·가족 리스크 없음 |
| +3 | +6 | **안전**: 부정 기사 10% 미만, 경미한 논란만 존재 |
| +2 | +4 | **양호**: 일부 리스크 있으나 치명적이지 않음 |
| +1 | +2 | **보통**: 일반적 수준의 리스크, 방어 가능 |
| -1 | -2 | **주의**: 부정 기사 20%+, 네거티브 소재 다수 |
| -2 | -4 | **위험**: 수사/의혹, 시민단체 지적, 방어 어려움 |
| -3 | -6 | **심각**: 기소/재판, 낙천 명단, 가족 비위 |
| -4 | -8 | **치명적**: 유죄 확정, 복합 리스크, 회복 불가 |
| X | 0 | **제외**: 리스크와 무관한 자료 |

---

## 5. X 판정 기준

**반드시 X:**
- 해당 후보와 무관한 부정 기사
- 동명이인 리스크
- 10년 이상 과거 사건 (시효 경과)

**X가 아닌 등급 처리:**
- 해당 후보의 부정 기사는 무조건 등급 부여 (-1 ~ -4)
- 리스크 없음을 확인하는 긍정 기사 → +1 ~ +4
- 경미한 논란 → -1

---

## 6. 판단 원칙

1. **역산 평가**: 리스크가 적은 후보가 높은 등급
2. **심각도 기준**: 법적 > 재산 > 시민단체 > 발언 > 정책 순
3. **경쟁자 비교**: 같은 그룹 내 리스크 수준 비교
4. **방어 가능성 반영**: 같은 리스크라도 대응 능력에 따라 차등
5. **rationale 한국어 1문장**
