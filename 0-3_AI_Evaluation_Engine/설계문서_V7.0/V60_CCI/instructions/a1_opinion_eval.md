## 1. 카테고리 정의

### 한글 정의
여론조사 추이와 바람 효과를 종합한 국민 지지 동향

### 영문 정의
Public support trends combining polling trajectories and bandwagon effects

### 학술 근거
- **Nate Silver (2012)**: 여론조사 가중 평균 모델 — 2012 대선 50개 주 정확 예측
- **강원택 (2010)**: 한국 "바람 효과" — 대통령 지지율 +1%p → 여당 후보 +0.3~0.5%p
- **허진재 (2022)**: 한국 여론조사 정확도 — 선거 1주 전 ±3%p
- **MBC 여론M**: 베이지안 통합 메타 여론조사 (370+ 조사 종합)

## 2. 평가 방식 — 상대평가 (플래툰 포메이션)

### 핵심 원칙
- **Claude 단독 평가** (소대장 조율로 일관성 보장)
- **같은 경쟁자 그룹 내 비교** (같은 선거구 출마 예상 후보)
- **레이팅 자체가 상대평가** — 별도 알고리즘 불필요

### 평가 맥락
```
같은 여론조사에서:
  1위 37.8% → 압도적 리드 → +4
  2위 23.7% → 14%p 격차 열세 → +2~+3
  3위 15.2% → 유의미한 격차 → +1
  4위 이하 → 군소 → -1 이하
```

## 4. 등급 체계 (+4 ~ -4) — 상대평가

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **압도적 1위**: 여론조사 1위, 격차 10%p+, 상승 모멘텀 |
| +3 | +6 | **안정적 선두**: 1위 또는 2위 초접전, 격차 5~10%p |
| +2 | +4 | **경쟁권**: 2~3위, 격차 5%p 이내, 바람 효과 긍정적 |
| +1 | +2 | **기본 경쟁력**: 인지도 있으나 상위권 아님, 횡보 |
| -1 | -2 | **하락세**: 지지율 하락 추세, 바람 효과 부정적 |
| -2 | -4 | **경쟁력 약화**: 부정 여론 우세, 격차 확대 |
| -3 | -6 | **심각한 열세**: 치명적 이벤트로 지지율 급락 |
| -4 | -8 | **당선 불가능**: 지지율 5% 미만, 군소 후보 수준 |
| X | 0 | **제외**: 여론조사 데이터와 무관한 자료 |

## 5. X 판정 기준 (제외)

**반드시 X 판정:**
- 해당 후보와 무관한 여론조사 데이터
- 선거구가 다른 여론조사
- 3년 이상 과거 여론조사 (시의성 상실)
- 동명이인 데이터

**X가 아닌 낮은 등급으로 처리:**
- 불리한 여론조사 결과 → -1 ~ -4
- 간접적으로 언급된 여론 → +1
- 부분적 여론 데이터 → +1 ~ +2

## 6. 판단 원칙

1. **경쟁자 그룹 내 비교**: 반드시 같은 선거구 후보와 비교하여 레이팅
2. **최신성 우선**: 최근 여론조사일수록 가중치 높게 판단
3. **추세 중시**: 단일 조사보다 추이(상승/하락/횡보) 중심 판단
4. **바람 효과 반영**: 대통령/정당 지지율 변동이 후보에 미치는 영향 고려
5. **rationale 한국어 1문장** (간결하게, 경쟁 맥락 포함)
