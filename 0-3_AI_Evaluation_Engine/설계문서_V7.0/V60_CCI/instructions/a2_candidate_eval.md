## 1. 카테고리 정의

### 한글 정의
현직 프리미엄, 인지도, 공천 상태 등 후보 개인의 구조적 경쟁 우위

### 영문 정의
Structural competitive advantage from incumbency premium, name recognition, and nomination status

### 학술 근거
- **Jacobson (2004)**: 현직자 효과(미국) — 현직 재선율 90%+, 득표 프리미엄 +5~8%p
- **황아란 (2016)**: 한국 현직자 효과 — 현직 국회의원 재선율 75~80%
- **박원호 (2011)**: 한국 현직자 프리미엄 +3~8%p
- **Lee (2008)**: 현직 프리미엄 +3~8%p

## 4. 등급 체계 (+4 ~ -4)

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **압도적 우위**: 현직 다선, 인지도 최고, 공천 확정, 조직 완비 |
| +3 | +6 | **강한 우위**: 현직 재선, 높은 인지도, 공천 유력 |
| +2 | +4 | **유리**: 현직 or 높은 인지도 도전자, 경선 우위 |
| +1 | +2 | **보통**: 1선 현직 or 인지도 있는 도전자 |
| -1 | -2 | **약세**: 도전자, 인지도 부족, 경선 경쟁 중 |
| -2 | -4 | **불리**: 정치 신인, 낮은 인지도, 공천 불확실 |
| -3 | -6 | **심각**: 공천 탈락 위기, 당내 기반 없음 |
| -4 | -8 | **최악**: 무명 후보, 조직 없음, 공천 불가 |
| X | 0 | **제외**: 후보자경쟁력과 무관한 자료 |

## 5. 현직 프리미엄 정량화

```
현직 효과 (한국 실증):
  국회의원 재선율: 75~80% [황아란(2016)]
  현직 프리미엄: +3~8%p [박원호(2011)]
  다선 프리미엄: 3선+ 추가 +2~3%p

도전자 핸디캡:
  초선 도전자: 기본 -5~-8%p 열세
  정치 신인: -10%p+ 열세
  무소속 도전자: -15%p+ 열세
```

## 6. 판단 원칙

1. **현직 여부가 핵심**: 현직 = 기본 +2, 도전자 = 기본 -1에서 시작
2. **인지도 보정**: 높은 인지도 도전자는 현직에 근접 가능
3. **공천 반영**: 공천 확정 = +1 보정, 경선 중 = 불확실성 반영
4. **경쟁자 비교**: 같은 그룹 내 후보별 구조적 우위 비교
5. **rationale 한국어 1문장**
