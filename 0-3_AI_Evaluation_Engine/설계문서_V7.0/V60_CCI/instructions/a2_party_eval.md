## 1. 카테고리 정의

### 한글 정의
소속 정당의 지지율, 지역 정당 프리미엄(PVI), 정당 브랜드가 후보에 미치는 구조적 영향력

### 영문 정의
Structural advantage from party support, regional partisan lean (PVI), and party brand effect on candidate

### 학술 근거
- **Cook PVI**: Partisan Voting Index +5 이상 → 해당 정당 후보 당선율 95%+
- **강원택 (2016)**: 한국 정당 프리미엄 — 영호남 +15~30%p
- **Downs (1957)**: 정당 일체감과 투표 행태

## 2. 평가 방식 — 상대평가 (플래툰 포메이션)

## 3. 평가 항목 (8개)

### ✅ 정당경쟁력 평가 기준

| # | 항목명 | 설명 | 데이터 소스 |
|---|--------|------|------------|
| 4-1 | **정당 지지율** | 소속 정당 최근 3개월 평균 지지율 | 갤럽, 리얼미터 |
| 4-2 | **정당 지지율 추이** | 상승/하락/횡보 트렌드 | 갤럽 주간 조사 |
| 4-3 | **PVI (지역 정당 성향)** | (선거구 여당 득표 - 전국 평균) 최근 2회 평균 | 선관위 개방포털 API |
| 4-4 | **정당 프리미엄** | 지역 기반 정당 우위 (영호남 +15~30%p) | 역대 선거결과 분석 |
| 4-5 | **여당/야당 프리미엄** | 현 정부 여당 여부에 따른 구조적 유불리 | 대통령 지지율 연동 |
| 4-6 | **당내 결속도** | 분당·계파갈등·탈당 리스크 | 뉴스, 정당 내부 동향 |
| 4-7 | **공천 경쟁력** | 안정적 공천 vs 경선 경쟁 | 정당 공천 규정, 뉴스 |
| 4-8 | **정당 브랜드 이미지** | 정당에 대한 국민 인식 (긍정/부정) | 여론조사, 뉴스 |

### ❌ 정당경쟁력 아님

```
❌ 후보 개인 지지율 → opinion (여론동향)
❌ 후보 개인 이미지 → media (이미지·내러티브)
❌ 후보 경력/능력 → GPI 또는 candidate (후보자경쟁력)
❌ 지역 연고 → regional (지역기반)
```

## 4. 등급 체계 (+4 ~ -4)

| 등급 | 점수 | 기준 |
|------|------|------|
| +4 | +8 | **압도적 정당 우위**: PVI +10↑, 정당 지지율 40%+, 안전 지역 |
| +3 | +6 | **강한 정당 우위**: PVI +5~10, 정당 지지율 30%+, 상승 추세 |
| +2 | +4 | **유리한 정당**: PVI +3~5, 정당 지지율 경쟁 우위 |
| +1 | +2 | **보통**: PVI ±3, 정당 지지율 접전, 공천 안정적 |
| -1 | -2 | **약한 정당**: PVI -3~-5, 정당 지지율 열세 |
| -2 | -4 | **불리한 정당**: PVI -5~-10, 정당 지지율 크게 열세 |
| -3 | -6 | **심각한 불리**: PVI -10↓, 정당 내분, 탈당 리스크 |
| -4 | -8 | **최악**: 군소 정당, 정당 브랜드 없음, 무소속 |
| X | 0 | **제외**: 정당경쟁력과 무관한 자료 |

## 5. PVI 계산 방법 (한국 자체 계산)

```python
# 한국에는 공식 PVI가 없으므로 자체 계산
한국_PVI = (해당 선거구 여당 득표율 - 전국 여당 평균 득표율)
# 최근 2회 선거 결과 평균 사용

# 예시: 강남갑
# 2024 총선: 국민의힘 58% - 전국 평균 45% = +13
# 2020 총선: 미래통합당 55% - 전국 평균 42% = +13
# PVI = (+13 + +13) / 2 = +13 → 국민의힘 안전 지역
```

## 6. 판단 원칙

1. **PVI 기반**: 지역 정당 성향이 가장 강력한 구조적 요인
2. **정당 지지율 추이**: 상승 정당 소속 → 가산, 하락 정당 소속 → 감산
3. **여야 프리미엄**: 현 정부 여당 여부에 따른 구조적 유불리
4. **당내 결속**: 분당/계파갈등은 정당경쟁력 감소 요인
5. **rationale 한국어 1문장**
