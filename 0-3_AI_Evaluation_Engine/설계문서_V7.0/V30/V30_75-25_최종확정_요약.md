```

    - 유튜브, 네이버 블로그, 인스타그램
    - 나무위키, 디시인사이드, 클리앙

  Perplexity (25개): 자체 검색엔진 (독립 크롤링)
    - 실시간 웹 크롤링 (매초 수만 건 인덱스 업데이트)
    - Google과 독립적인 검색 결과
    - 다양한 소스 발견 가능
```
