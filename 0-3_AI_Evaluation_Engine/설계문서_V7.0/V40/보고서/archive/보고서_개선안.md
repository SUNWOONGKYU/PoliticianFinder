## 현재 보고서 분석

### 예상되는 일반적인 문제점

**1. 가독성 문제**
- 긴 텍스트 블록으로 인한 피로감
- 핵심 메시지 파악 어려움
- 데이터와 해석의 혼재

**2. 데이터 시각화 부족**
- 숫자 나열식 표현
- 점수 간 비교 어려움
- 추세 파악 불가

**3. 전문성 vs 접근성 균형**
- 과도하게 기술적이거나
- 지나치게 단순화되거나

**4. 사용자 경험**
- 원하는 정보 찾기 어려움
- 모바일 최적화 미흡
- 인쇄/공유 불편

## 핵심 개선 방향

### 1. 계층적 정보 구조

```
한눈에 보는 요약 (Executive Summary)
  ↓
핵심 지표 대시보드 (Key Metrics)
  ↓
카테고리별 상세 분석 (Deep Dive)
  ↓
데이터 근거 (Evidence)
  ↓
부록 (Appendix)
```

**효과**:
- ✅ 원하는 깊이만큼 읽기 가능
- ✅ 바쁜 사용자는 요약만
- ✅ 관심 있는 사용자는 상세까지

### 2. 카피라이팅 원칙

**명확성 (Clarity)**
- 전문 용어 최소화
- 필요시 즉시 설명
- 한 문장 = 하나의 메시지

**임팩트 (Impact)**
- 핵심 먼저, 세부사항 나중
- 강조 포인트 명확화
- 액션 가능한 인사이트

**일관성 (Consistency)**
- 동일 개념은 동일 용어
- 톤앤매너 통일
- 브랜드 보이스 유지

### 3. 시각화 우선

**데이터 → 비주얼**
```
점수 800점 (Good 등급)
  ↓
[█████████░] 80%
S - B - G - P - E - D - M
        ↑ 현재 위치
```

**비교 → 그래프**
```
전문성  ████████░░ 82점
리더십  ███████░░░ 73점
청렴성  █████████░ 91점
```

## 카피라이팅 개선안

### 1. 헤드라인 & 서브헤드

#### ❌ 개선 전 (약함)

```
"박주민 의원 평가 결과"
"점수: 800점"
"전문성이 높음"
```

**문제점**:
- 밋밋함, 임팩트 없음
- 숫자만 나열
- 감정적 연결 실패

#### ✅ 개선 후 (강함)

```
"법률 전문가의 진면목, 시민 곁으로 한 걸음 더"
→ 강점 강조 + 개선 방향 제시

"상위 15% 전문성, 하지만 소통은 과제"
→ 긍정 + 균형 + 구체적 수치

"변호사 출신 의정활동, 숫자로 증명하다"
→ 배경 + 성과 연결
```

**개선 원칙**:
- ✅ 임팩트: 숫자 + 맥락
- ✅ 균형: 강점 + 개선점
- ✅ 구체성: 추상적 표현 최소화

### 4. 톤앤매너

#### 전문성 + 접근성 균형

**전문적 (Professional)**
- 데이터 근거 명시
- 출처 투명 공개
- 객관적 표현 사용

**친근한 (Approachable)**
- 전문 용어 즉시 설명

- 대화체 문장 구조

**예시**:

✅ "청렴성, 상위 5%.
윤리 기준에서 거의 만점에 가까운 평가입니다."
→ 전문성 유지 + 친근한 해석

#### 점수 바 (Score Bar)

```
전문성  ████████░░ 87/100
리더십  ███████░░░ 73/100
비전    ██████░░░░ 62/100
청렴성  █████████░ 91/100
```

**코드 예시**:
```python
def generate_bar(score, max_length=10):
    filled = int(score / 10)
    empty = max_length - filled
    return '█' * filled + '░' * empty

print(f"전문성  {generate_bar(87)} 87/100")
```

### 강력한 동사 리스트

**평가/분석**
- 입증하다, 증명하다, 확인하다
- 보여주다, 드러내다, 나타내다

**성과/결과**
- 달성하다, 이루다, 성취하다
- 통과시키다, 실현하다, 완수하다

**개선/변화**
- 개선하다, 강화하다, 향상하다
- 확대하다, 증가하다, 상승하다

**문제/과제**
- 필요하다, 요구되다, 과제로 남다
- 개선이 필요하다, 보완해야 한다
