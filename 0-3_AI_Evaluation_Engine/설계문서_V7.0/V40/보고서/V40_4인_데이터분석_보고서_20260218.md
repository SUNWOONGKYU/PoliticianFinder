### 1.2 V40 평가 시스템 개요

V40는 4개 AI가 독립적으로 정치인을 평가하는 앙상블 평가 시스템이다.

**수집 채널:** Gemini CLI + Naver API (각 50% 비중)
**평가 AI:** Claude Haiku 4.5 / ChatGPT gpt-5.1-codex-mini / Gemini 2.0 Flash / Grok 3
**평가 카테고리:** 전문성, 리더십, 비전, 청렴성, 윤리성, 책임감, 투명성, 소통능력, 대응성, 공익성 (10개)
**등급 체계:** +4 / +3 / +2 / +1 / -1 / -2 / -3 / -4 (8단계), X = 평가 제외
**점수 공식:**

```
score = rating × 2  (+4 → 8점, +1 → 2점, -1 → -2점, -4 → -8점)
category_score = (6.0 + avg_score × 0.5) × 10  (범위: 20~100점)
final_score = round(min(Σ 10개 카테고리, 1000))  (범위: 200~1000점)
```

**버퍼:** 카테고리당 기본 100개 수집, 최대 120개 (20% 버퍼)

### 1.3 분석 목적 및 방법론

본 보고서는 V40 평가 완료 후 다음 세 가지 관점에서 결과를 검토한다.

1. **내적 타당성:** AI 간 평가 순위가 얼마나 일치하는가 (Spearman 순위상관계수)
2. **신뢰도:** 어떤 카테고리에서 AI 간 이견이 크고 작은가
3. **외적 타당성:** AI 평가 결과가 시민 여론(지지율)과 어떤 관계를 보이는가

## 5. 평가 신뢰도 종합 평가

### 5.1 신뢰도 4개 축 분석

| 축 | 측정 방법 | 결과 |
|----|---------|------|
| 순위 신뢰도 | AI 간 Spearman ρ | 종합 0.95 / 카테고리 평균 0.81 |
| 절대점수 일관성 | AI별 평균점수 편차 | Gemini +44점 편향 존재 |
| 카테고리별 안정성 | 카테고리별 ρ 분포 | 0.50 ~ 0.95 (혼재) |
| 통계적 한계 | 샘플 크기 | n=4 (소수) |

### 5.2 강점 (신뢰도 높은 측면)

**① 종합 순위는 강하게 신뢰할 수 있음 (ρ = 0.95)**
- 4개 독립 AI가 모두 "정원오 1위, 오세훈 4위"에 동의
- 서로 다른 회사, 다른 학습 데이터, 다른 알고리즘에도 불구하고 동일 결론

**② 평균화(앙상블)가 노이즈를 상쇄**
- 카테고리 단위 ρ = 0.81 → 종합 ρ = 0.95로 상승
- 4개 AI 평균을 최종 점수로 사용하는 구조가 개별 AI 오류를 보정

**③ 리더십·책임감·투명성·청렴성·윤리성은 AI 간 강한 합의 (ρ ≥ 0.90)**

### 5.3 약점 및 주의사항

**① Gemini 관대 편향 (+44점)**

| AI | 4명 평균점수 |
|----|:-----------:|
| Gemini | 783점 |
| ChatGPT | 753점 |
| Claude | 740점 |
| Grok | 738점 |

→ 절대점수를 AI별 단독 인용 금지. **반드시 4 AI 앙상블 평균 사용**

**② 대응성·전문성·비전 카테고리 불안정 (ρ = 0.50~0.65)**
- AI마다 순위가 뒤바뀌는 현상 발생
- 이 3개 카테고리의 단독 해석은 위험

**③ 2-3위(박주민 vs 조은희) 통계적 불확실성**
- 평균 10점 차이 → 오차 범위 내 동등 가능성
- "박주민이 조은희보다 낫다"는 확정 불가

**④ n=4 소수 샘플 한계**
- Spearman ρ의 통계적 유의성(p-value)이 n=4에서 낮음
- 정치인 수가 늘수록 신뢰도 판단의 정밀도 상승

### 5.4 신뢰도 등급표

| 사용 목적 | 신뢰도 | 판정 |
|---------|:-----:|------|
| 1위 vs 4위 구분 | ★★★★★ | 완전 신뢰 |
| 전체 순위 | ★★★★☆ | 신뢰 (2-3위 오차 주의) |
| 카테고리별 순위 | ★★★☆☆ | 조건부 신뢰 (카테고리에 따라 다름) |
| 절대점수 단독 인용 | ★★☆☆☆ | AI별 편향 존재, 주의 필요 |
| 4 AI 평균점수 | ★★★★☆ | 신뢰 (편향 상쇄됨) |

### 5.5 한계 및 시사점

- V40는 **뉴스·보도 기반** 평가로, 언론 노출이 적은 정치인은 데이터 부족 가능성
- 수집 기간(OFFICIAL 4년, PUBLIC 2년) 외 이전 행적은 반영 안 됨
- 동일 기사를 4 AI가 모두 수집하면 풀에 4번 포함되어 가중치 효과 발생 (의도된 설계)
- 시계열 변화(과거 vs 현재)는 현재 버전에서 미반영

### 6.3 일치·역전 사례 분석

**[일치] 정원오 — 가장 일관된 평가**

- **해석:** 실적 기반 평가와 여론이 일치하는 이상적 사례

**[일치] 박주민 — 3위로 수렴**
- V40 3위(782점) + 지지율 여권 2위(8%)
- 여권 후보군 내에서 상대적 위치가 일치
- 다만 절대 지지율이 정원오 대비 매우 낮아 당선 가능성에서 큰 격차

**[역전 1] 오세훈 — V40 꼴찌, 지지율 1위**
- V40: 청렴성 66점·윤리성 66점 (4명 중 공동 최하위)
- 지지율: 24~29.4% (전체 1위)
- **역전 원인:**
  - 현직 서울시장 5선 도전 → 현직 프리미엄 + 압도적 인지도
  - 보수층 결집 + 20·30·60대 이상에서 강세
  - 탄핵 찬성으로 친윤 이미지 탈피 → 중도층 흡수
- **해석:** AI는 도덕성·실적 기반 평가, 유권자는 인지도·정당·정치 환경 기반 지지

**[역전 2] 조은희 — V40 2위, 지지율 최하위(2%)**
- V40: 783점 E등급 (정원오와 1점 차)
- 지지율: 야권 후보군 중 5위(2%)
- **역전 원인:**
  - 국민의힘 소속 → 서울 전체 여론(진보 유권자 포함)에서 불리
  - 오세훈이라는 현직 프리미엄 주자 존재 → 야권 내 묻힘
  - 국회의원(서초갑)으로 전시 인지도 낮음
- **해석:** AI는 의정 활동 기준으로 높게 평가하나, 지지율은 당내 경쟁 구도와 인지도에 좌우됨

### 6.4 Spearman ρ (V40 vs 지지율)

V40 순위: 정원오(1) > 조은희(2) > 박주민(3) > 오세훈(4)
지지율 순위: 오세훈(1) > 정원오(2) > 박주민(3) > 조은희(4)

```
d 값:
  오세훈: 4-1 = 3  → d² = 9
  정원오: 1-2 = -1 → d² = 1
  박주민: 3-3 = 0  → d² = 0
  조은희: 2-4 = -2 → d² = 4
  Σd² = 14

ρ = 1 - 6×14 / (4×15) = 1 - 84/60 = -0.40
```

**V40 vs 지지율 Spearman ρ = −0.40 (약한 역상관)**

> 완전히 반대 방향도 아니고, 일치하지도 않는다.
> 박주민의 3위 일치가 상쇄 역할을 하며, 오세훈·조은희의 역전이 부정적 방향을 만든다.

### 6.5 해석: V40와 지지율이 측정하는 것의 차이

| 구분 | V40 평가 | 지지율 |
|------|---------|-------|
| 측정 대상 | 실적·정책·도덕성 (AI 분석) | 인지도·정당·선거 경쟁력 (시민 여론) |
| 오세훈 저평가 이유 | 청렴·윤리 관련 논란 반영 | 현직 프리미엄으로 차단됨 |
| 조은희 고평가 이유 | 의정 활동 객관 평가 | 진보 유권자 포함 전체 여론서 불리 |
| 정원오 일치 이유 | 구체적 행정 성과 존재 | 실적이 여론에도 전달됨 |

## 7. 종합 결론

### 7.1 V40 시스템의 의의

**① 평가 신뢰도 확인**
- 4개 독립 AI의 종합점수 순위 합의도 ρ = 0.95로 매우 높음
- 카테고리 단위에서도 평균 ρ = 0.81 수준 유지
- 1위(정원오)·4위(오세훈) 합의는 완벽 (ρ = 1.00)
- **결론:** V40는 다수 AI 앙상블 방식으로 신뢰할 수 있는 순위를 도출함

**② 지지율과의 차별화된 가치**
- V40 점수와 여론 지지율의 ρ = -0.40 (약한 역상관)
- 이는 V40가 지지율을 예측하는 도구가 아님을 의미
- **V40의 가치:** 인지도·정당에 가려진 실제 역량을 드러내는 객관적 지표
- 특히 오세훈(지지율 1위, V40 4위)의 역전 현상이 이를 명확히 보여줌

**③ 정원오의 특수성**
- V40와 지지율 모두에서 상위권 → **실적과 여론이 일치하는 정치인**
- 행정 성과가 AI 평가와 시민 지지율 양쪽에서 검증된 사례
