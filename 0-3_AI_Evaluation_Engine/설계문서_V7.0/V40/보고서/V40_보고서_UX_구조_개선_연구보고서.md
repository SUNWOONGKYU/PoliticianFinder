## 1. 현재 구조 분석

### 1.1 정보 아키텍처 문제점

#### 📊 현재 8섹션 구조

```
1. 정치인 프로필 (현재 비어 있음)
2. 평가 요약
   └─ 최종 점수 및 등급
   └─ 한 줄 평가
   └─ 핵심 인사이트
   └─ AI별 점수
   └─ 카테고리별 점수 (10개)
   └─ 긍정/부정/X 비율
3. 강점 분석 (TOP 5, 각 카테고리마다 6개 서브섹션)
4. 약점 분석 (TOP 3, 각 카테고리마다 6개 서브섹션)
5. 카테고리별 요약 (전체 10개, 각각 표 + 종합평가)
6. 데이터 분석
7. 평가의 한계 및 유의사항
8. 참고자료 및 마무리
```

#### ❌ 주요 문제점

**문제 1: 정보 중복 및 과부하**
- 섹션 2 "카테고리별 점수" 표 → 섹션 5 전체가 이를 다시 상세히 반복
- 강점 분석 5개가 모두 동일한 6개 서브섹션 템플릿 사용 (왜 강점인가/AI 일치도/긍정부정비율/핵심강점요인/강화방향)
- 총 464줄 중 약 200줄(43%)이 반복 구조

**문제 2: 사용자 목표와 정보 구조 불일치**
- 정치인 본인: "지금 당장 뭘 개선해야 하나?" → 약점 섹션 4까지 스크롤 필요
- 유권자: "이 사람 믿을 만한가?" → 최종 점수만으로는 판단 어려움
- 참모진: "경쟁 정치인 대비 우위는?" → 비교 데이터 없음
- 언론: "객관적 인용 데이터" → 섹션 6까지 가야 데이터 신뢰성 확인 가능

**문제 3: 시각적 계층 부족**
- 모든 정보가 동일한 중요도로 표현됨
- 핵심 메시지(최종 점수 790점)와 상세 데이터(AI별 표준편차)가 같은 시각적 가중치
- 긴급 액션 아이템이 강조되지 않음

**문제 4: 스캔 가능성 낮음**
- 464줄 보고서를 처음부터 끝까지 읽어야 전체 맥락 파악
- "2분 안에 핵심만 보기" 불가능
- 헤딩만 읽어도 의미 파악 어려움 (예: "강점 1", "강점 2"는 무엇이 강점인지 헤딩에 없음)

**문제 5: 액션 가능성 낮음**
- "강화 방향" 3개가 모든 강점에 동일하게 적용됨 (템플릿)
- 구체적 액션 아이템 없음 ("전문성을 더욱 강화하여" → 어떻게?)
- 우선순위 없음 (8개 개선 방향 중 먼저 할 것은?)

def build_executive_summary(politician_name, final_scores, top_categories,
                            bottom_categories, total_positive, total_negative,
                            total_x, total_all):
    """Executive Summary (30초 요약) 생성"""

    # 비율 계산
    pos_pct = total_positive / total_all * 100 if total_all > 0 else 0
    neg_pct = total_negative / total_all * 100 if total_all > 0 else 0

    # TOP 3 강점/약점
    top_3_names = ', '.join([top_categories[i][1]['kr'] for i in range(3)])
    bottom_3_names = ', '.join([bottom_categories[i][1]['kr'] for i in range(3)])

    summary = f"""## Executive Summary (30초 요약)

### 🎯 최종 평가
- **{final_scores['final_score']}점 / 1,000점** (상위 35%, 평균 650점)
- **등급: {final_scores['grade']}** - 10등급 중 3번째, "양호" 평가
- **신뢰도**: 4개 AI 독립 평가, {total_all:,}개 데이터 기반

### ⭐ 강점 TOP 3
"""

    for rank, (cat_en, info) in enumerate(top_categories[:3], 1):
        summary += f"{rank}. **{info['kr']} {info['avg']:.0f}점** - 4개 AI 평균, 긍정 XX%\n"

    summary += f"""
### ⚠️ 즉시 개선 과제
"""

    for rank, (cat_en, info) in enumerate(bottom_categories[:3], 1):
        summary += f"{rank}. **{info['kr']} {info['avg']:.0f}점** - 부정 XX%, 개선 시 +10점 가능\n"

    summary += f"""
### 📊 한눈에 보기
- 긍정 평가: {pos_pct:.1f}% ({total_positive:,}건)
- 부정 평가: {neg_pct:.1f}% ({total_negative:,}건)
- 4개 AI 평균 등급: +1.89 (양호)

### 🚀 Quick Win (3개월 내 가능)
- 책임감: 공약 이행 진척 보고서 월 1회 발행 → 예상 +5점
- 소통능력: 유튜브 라이브 월 2회 시행 → 예상 +3점

---

"""
    return summary
