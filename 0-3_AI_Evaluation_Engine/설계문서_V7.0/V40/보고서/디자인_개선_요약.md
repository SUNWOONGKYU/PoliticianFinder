### 개선 6: 인쇄 최적화

**A4 페이지 설정**
```css
@page {
  size: A4 portrait;
  margin: 20mm 25mm 20mm 25mm;  /* 상 우 하 좌 */

  /* 헤더 (상단 중앙) */
  @top-center {
    content: "조은희 의원 V40 평가보고서";
  }

  /* 페이지 번호 (하단 중앙) */
  @bottom-center {
    content: "- " counter(page) " -";
  }
}
```

**페이지 브레이크 제어**
```css
/* 제목 뒤에서 페이지 넘기지 않기 */
h1, h2, h3, h4 {
  page-break-after: avoid;
}

/* 표, 카드 중간에서 페이지 넘기지 않기 */
table, .score-card, .strength-card, .weakness-card {
  page-break-inside: avoid;
}
```

**배경색 인쇄 강제**
```css
@media print {
  * {
    -webkit-print-color-adjust: exact !important;
    print-color-adjust: exact !important;
  }
}
```

**효과**:
- 펀칭 구멍 고려 (여백)
- 제본 가능 (좌측 25mm)
- 섹션이 자연스럽게 이어짐
- 색상 그대로 인쇄

## 4. 핵심 디자인 철학

### 4-1. 신뢰와 전문성 (파란색)

**왜 파란색인가?**
- 정치 보고서 국제 표준 (UN, NATO, EU)
- 색채 심리: 신뢰, 안정, 지성
- 인쇄 시 가독성 우수 (흑백에서도 명확)

**어디에 사용?**
- H1, H2 제목 (진한 파란)
- 표 헤더 배경 (진한 파란)
- 테두리, 링크 (밝은 파란)
- Executive Summary 박스 (연한 파란 배경)

---

### 4-2. 명확한 구분 (초록/주황)

**강점 = 초록색**
- 긍정적 감정
- "계속하세요" 신호
- 국제 표준 (교통 신호)

**약점 = 주황색**
- 주의 필요
- 빨강보다 덜 자극적
- "개선하세요" 신호

**효과**:
- 3초 안에 강점/약점 파악
- 감정적 연결 (색상 = 의미)
- 실행 촉구 (신호등 효과)

---

### 4-3. 정보 위계 (크기, 굵기, 색상)

**레벨 1: 최우선 (즉시 눈에 띄어야 함)**
- 최종 점수 (816점) → 36pt, 굵게, 파란색
- 등급 (E) → 24pt, 파란 배경, 흰색 글자
- H1 제목 → 32pt, 굵게, 파란색

**레벨 2: 중요 (스캔 시 파악)**
- H2 섹션 제목 → 24pt, 파란색, 좌측 테두리
- 점수 (86점, 84점...) → 11pt, 굵게
- 배지 (⭐ 최고, ⚠️ 개선 필요) → 작지만 색상 강조

**레벨 3: 본문 (자세히 읽을 때)**
- 설명 텍스트 → 11pt, 일반 굵기
- 리스트 → 약간 들여쓰기
- 메타 정보 → 9pt, 회색

**효과**:
- 바쁜 정치인도 3분 안에 핵심 파악
- 자세히 읽고 싶으면 본문으로
- 정보 과부하 방지

---

### 4-4. 실용성 (읽고 → 행동)

**단순 평가가 아닌 개선 도구**
- 강점 → "강화 방향" 제시
- 약점 → "개선 방향" 제시
- 체크리스트 → 실행 가능한 항목

**예시**:
```
약점 1: 청렴성 논란 (76점)

개선 방향 🎯
1. ☐ 투명한 인사 프로세스 공개
2. ☐ 제3자 감사 요청
3. ☐ 정기적 윤리 교육 이수 (분기별)
```

**효과**:
- 보고서를 읽고 실제 행동으로 이어짐
- 체크박스로 진행 상황 추적
- 구체적 실행 방법 제시

2. 개선하고 싶으니까 → **실행 가능한 방향 제시** (강화 방향, 개선 방향)
3. 공유하고 싶으니까 → **인쇄 가능한 품질** (A4, 페이지 번호, 헤더)

→ **사용자 니즈 중심 설계**
