# Category 08: Communication (소통능력) - 평가 지침서 V40

---

## 1. 카테고리 정의

### 한글 정의
시민과 효과적으로 의사소통하는 능력

### 영문 정의
The ability to communicate effectively with citizens

### 학술 근거
- **Habermas (1984)**: 의사소통적 합리성 이론
- **Fishkin (2009)**: 숙의 민주주의와 소통
- **Nabatchi (2012)**: 시민 참여와 소통의 관계


## 2. V40 평가 방식 - 풀링 평가

### 평가 대상 데이터

수집 단계에서 collected_data_v40 테이블에 저장된 풀링 데이터 전체를 평가합니다.

4개 평가 AI (Claude Haiku 4.5, ChatGPT gpt-5.1-codex-mini, Gemini 2.0 Flash, Grok 3)가
각각 전체 풀링 데이터를 독립적으로 평가합니다.

### 평가 독립성

```
타이밍 분리 = 객관성 보장

[평가 단계] - 수집과 다른 세션, 다른 시점
→ Claude (Haiku 4.5), ChatGPT (gpt-5.1-codex-mini), Gemini (2.0 Flash), Grok (3)이 평가
→ 수집 시점 ≠ 평가 시점
→ 독립적 판단 = 더 객관적
```

## 4. 등급 체계 (+4 ~ -4) - V40

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 소통의 달인, 시민과 공감, 다채널 활용 |
| **+3** | +6 | 우수 - 적극적이고 효과적인 소통, SNS 활발 |
| **+2** | +4 | 양호 - 소통 노력 인정됨, 기본적 채널 운영 |
| **+1** | +2 | 경미한 긍정 - 기본적인 소통, 특별한 노력 없음 |

| **-1** | -2 | 경미한 부정 - 소통 부족 지적, 채널 운영 소홀 |
| **-2** | -4 | 일반 부정 - 소통 능력 문제, 시민과 거리감 |
| **-3** | -6 | 심각한 부정 - 불통, 소통 채널 부재 |
| **-4** | -8 | 최악 - 완전한 불통, 모든 소통 거부 |

| **X** | 0 | 평가 제외 - 10년 이상 과거, 동명이인, 무관 내용, 날조 정보 |

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 소통의 달인으로 인정
- 시민과 적극적 공감
- 다양한 채널 효과적 활용
- 토론/설득 능력 탁월

### +3등급 (우수함)
- 적극적이고 효과적인 소통
- SNS 등 채널 활발 운영
- 시민 의견 경청
- 언론 대응 우수

### +2등급 (양호함)
- 소통 노력 인정됨
- 기본적인 채널 운영
- 시민 의견 수렴
- 큰 문제 없음

### +1등급 (경미한 긍정)
- 기본적인 소통
- 특별한 노력 없음
- 평균 수준
- 무난한 평가

### -1등급 (경미한 부정)
- 소통 부족 지적
- 채널 운영 소홀
- 시민 의견 경청 부족
- 개선 필요

### -2등급 (일반 부정)
- 소통 능력 문제
- 언론 대응 미흡
- 시민과 거리감
- 불통 비판

### -3등급 (심각한 부정)
- 불통
- 소통 채널 부재
- 시민 의견 무시
- 심각한 소통 문제

### -4등급 (최악)
- 완전한 불통
- 모든 소통 거부
- 시민과 단절
- 정치인 부적합

## 6. 평가 프로세스

### 입력 데이터
```
해당 카테고리의 풀링 데이터 전체
(collected_data_v40 테이블에서 조회)
```

### 평가 방법
1. **카테고리 풀링 데이터 전체 검토** (수집된 모든 데이터)
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 채널과 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **일방 vs 쌍방**: 일방적 홍보 vs 실질적 소통
2. **양 vs 질**: SNS 게시물 수 ≠ 소통 품질
3. **선별 소통**: 특정 집단만 소통하는지 확인
4. **대리 소통**: 본인 소통 vs 보좌관 대리 소통
