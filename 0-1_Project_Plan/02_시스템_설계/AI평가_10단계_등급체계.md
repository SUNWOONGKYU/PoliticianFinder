## ⚠️ 주의사항

### 등급 표시 원칙
1. **점수 우선**: 점수를 먼저 보여주고 등급은 보조 정보
2. **색상 일관성**: 같은 등급은 항상 같은 색상 사용
3. **접근성**: 색상만으로 구분하지 않고 텍스트/이모지 병행

### 등급 업데이트 정책
- 점수가 변경되면 등급도 자동 업데이트
- 실시간 반영 (캐시 주의)
- 등급 변경 시 사용자에게 알림 (선택사항)

### 사용자 오해 방지
- "Prior 7.0 = 700점"이 기본값임을 명시
- 데이터 부족 시 등급 미표시 또는 "평가 중" 표시
- 등급 산정 기준을 투명하게 공개
