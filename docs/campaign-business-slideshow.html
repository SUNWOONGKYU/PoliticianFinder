06 유세 동선 지도 — 생성 프로세스

PoliticianFinder가 후보자 이름 하나만 입력받으면, AI가 자동으로 최적 동선을 생성합니다.

Step 1 · 데이터 수집
선관위 API에서 실제 선거 데이터 자동 수집
투표구별 득표수·투표율·유권자수 → 선거구별 경합도 원천 데이터 확보
Step 2 · AI 경합도 분석
4개 AI가 선거구별 전략적 가치를 정량화
경합도(40%) + 유권자규모(30%) + 투표율(20%) + AI평가(10%) → 우선순위 점수 산출
Step 3 · 78슬롯 최적 배분
13일 × 6시간대 = 78개 슬롯에 지역구 자동 배정
경합 지역 50% · 우세 지역 30% · 열세 지역 15% · 포기 지역 5% 시간 배분
Step 4 · 인터랙티브 지도 생성
일자별·시간대별 동선을 지도 위에 시각화
Leaflet.js 기반 자동 생성 → 클릭 한 번으로 일자별 동선 확인 + 자동 재생 시뮬레이션

소요 시간: 후보자 이름 입력 → 완성된 동선 지도 30분 이내 자동 생성

06 왜 이 전략의 수준이 다른가

단순한 지도가 아닙니다. AI 평가 데이터가 전략 수립에 직접 반영되기 때문에, 다른 솔루션과는 근본적으로 품질이 다릅니다.

일반 동선 솔루션

  • 투표율·인구수만 보고 지역 배분
  • 후보자 성향·강약점 반영 불가
  • 경합 지역 판단 = 감(感)
  • 선거 때마다 처음부터 다시

PoliticianFinder 기반 전략

  • 4개 AI × 10개 카테고리 평가 데이터 반영
  • 후보 강점 지역에 시간 집중 배분 자동
  • 경합도 = 정량적 데이터 기반
  • 71명+ 누적 분석 → 패턴 학습

핵심: 지도는 누구나 만들 수 있지만, 어디에 얼마나 시간을 써야 하는지를 정확히 아는 것은
수년간 축적된 AI 평가 데이터와 분석 노하우가 있어야만 가능합니다.
이 데이터 없이 만든 동선은 "예쁜 지도"일 뿐, "이기는 전략"이 아닙니다.
