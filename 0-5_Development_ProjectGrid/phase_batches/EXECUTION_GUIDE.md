## 🚀 실행 방법

### 1단계: Phase 1 실행

```
1. Phase_1_batch.txt 파일 열기
2. 전체 내용 복사
3. Claude에게 붙여넣기
4. 완료 대기 (약 20~30분 예상)
```

### 2단계: Phase 2 실행

Phase 1 완료 후:
```
1. Phase_2_batch.txt 파일 열기
2. 전체 내용 복사
3. Claude에게 붙여넣기
4. 완료 대기
```

### 3~7단계: 나머지 Phase 실행

동일한 방법으로 Phase 3~7 순차 실행

## ⚠️ 주의사항

1. **순서 준수**: Phase 1 → 2 → ... → 7 순서대로 실행
2. **완료 확인**: 각 Phase 완료 후 다음 Phase 시작
3. **오류 처리**: 오류 발생 시 해당 작업 건너뛰고 계속 진행
