3D 좌표계 (X축=Phase, Y축=Area, Z축=Number)

규칙:
  ✓ X/Y/Z 중 하나라도 변화 → [기존ID]_[새로운ID]
  ✓ 변화 없음 → [기존ID] (유지)

예시:
  P0BA0 → P2BA0 (X축: 0→2)      → ID 체인: P0BA0_P2BA0
  P1F1  → P1F1  (변화 없음)      → ID 체인: P1F1 (유지)
  P7D1  → P3D1  (X축: 7→3)      → ID 체인: P7D1_P3D1
