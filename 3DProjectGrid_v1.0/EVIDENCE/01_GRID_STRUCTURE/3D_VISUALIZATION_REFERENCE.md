## 📊 시각화 내용

### 3D 그리드 구조 표현

이 HTML 파일은 다음을 시각적으로 표현합니다:


### 인터랙티브 기능

- ✅ **3D 회전**: 마우스로 그리드 회전 가능
- ✅ **확대/축소**: 스크롤로 확대/축소
- ✅ **작업 상세**: 각 Task 클릭 시 정보 표시
- ✅ **필터링**: Phase/Area 별 필터링 가능
- ✅ **상태 표시**: 색상으로 작업 상태 표현 (완료/대기/진행)
